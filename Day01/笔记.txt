1.java语言的特点
	开源: 利于发展, 利于成长
	跨平台: jvm可以实现扩平台
2.为了我们需要配置环境变量
	我们可以在任何地方都可以使用jdk的工具, java命令, javac命令
3.入门案例(重点!!!!!!)
	步骤:
		1.编写源代码
			源代码必须放在后缀名为.java的文件中, 必须声明一个类, 写上主方法
			例如:
				public class HelloWorld{
					public static void main(String[] args){
						System.out.println("HelloWorld");
					}
				}
		2.编译源代码成字节码
			格式: javac 源码文件名.java
			其实就是讲 .java的文件翻译成.class文件,原因是jvm只认识.class的文件
		3.运行字节码
			java的命令来运行
			格式:
				java  字节码名(不需要后缀名)
	常见问题:
		1.大小写问题
		2.中文字符问题
		3.main写了mian, 没有主方法方法的问题
		4.后缀名的问题
4.关键字(重点!!!!!!)
	被java赋予特殊含义的单词
	public ,class , static , void , byte, short, int , long, char, boolean , float , double
	特点:
		1.java中关键字都是小写的
		2.高级开发工具都有高亮显示
5.常量(重点!!!!!!)
	整数常量:
		包含了所有的整数
	小数常量
		包含所有的小数
	字符常量
		特点:必须用单引号引起来, 单引号有且只能有一个字符,例如'a'
	字符串常量
		特点: 必须用双引号一起来 "我好开心认识你们"
	boolean常量
		true,false
	null常量
		引用数据类型的时候再去讲解
6.变量(重点!!!!!!)
	格式:
		数据类型 变量名 = 值;
		数据类型: 定义变量的范围
		变量名: 用来操作变量的
		值: 就是存储在变量中的数据
	本质: 其实就是一个容器, 专门用来存放数据, 但是这个容器比较小, 只能存放一个数据!!!!!
	
		
	数据类型:
		基本数据类型:
			四类八种
				整数: byte,short,int,long
				小数: float,double
				boolean类型: true和false
				char:
		引用数据类型(后期会讲解)
	变量在定义时注意事项
		1.整数默认是int类型的,小数默认是double类型,定义float的时候注意加F, 定义Long类型的话注意加L
		2.变量参数运算是其值在参与运算
		3.变量未赋值，不能使用.(因为变量参与运算是其值参与运算,没有值怎么参与运算)
		4.变量只在它所属的范围内有效.(变量在其所属的大括号中有效, 出了大括号就会消失);
		5.一行上可以定义多个变量，但是不建议
		6.java中存在常量优化机制
		7.同一个作用范围不能出现同名的变量
7.标识符
	我们给类, 接口, 方法, 变量起的名字
	规则:
		1.必须是52个字母, _,$数字
		2.数字不能开头
		3.不能是java的关键字
	规范:
		1.类,接口,枚举,注解  ---> 要符合大驼峰式, 每个单词的首字母都大写
		2.方法,变量 --->要符合小驼峰式, 从第二个单词首字母大写
		3.等到讲解final关键字时再说
		4.包 ---> 建议全部小写, 多个包之间用"."隔开, 而且包名建议域名倒写, www.baidu.com  : com.baidu
